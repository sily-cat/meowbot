meowbot

a kinda useless discord bot

created by @silyduct, hosted on deno deploy

!!disclaimer!! - the account sily.cat was just falsely suspended by discord, so that's why the bot is now called meowbot-ii

source code

current features:

/say - say a message (as a response to the slash command)

/send - say a message (as the bot itself, only in guilds, requires "meowbot send" role)

/cat - send a cat image from thecatapi (with a button to send another)

/meow - send a random cat noise from a list (dm me on discord to suggest more!)

/ping - ping the bot and get the deno deploy server it's running on

/octad [generate, solve]* - something kinda complicated i may explain later

/catness [user] - gets the % cat someone is

/wipers - returns whether windshield wipers are on sale today or not
